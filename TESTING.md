## Запуск приложения

```bash
cd healthfull
pnpm dev
```

Откройте: http://localhost:3000

## Как играть

1. **Главная страница** → Click "Start Game"
2. **Ввод имени** → Введите ваше имя
3. **Смотрите видео** → Футбольное видео начнет играть
4. **Угадайте момент гола** → Click "Guess Now!" на 3-й секунде
5. **Получите результат** → Увидите:
   - Ваше время клика
   - Разница (delta) в миллисекундах  
   - Очки (score)

## Система очков

| Разница | Очки |
|---------|------|
| ≤ 300ms | 100  |
| ≤ 1s    | 70   |
| ≤ 2s    | 40   |
| ≤ 5s    | 10   |
| > 5s    | 0    |

## Идеальный результат

Кликните в диапазоне **2700ms - 3300ms** = **100 очков!** 🎯
